## Format Rules

- Timestamp in chapter headers: `[M:SS]` or `[H:MM:SS]` for videos over 1 hour
- Timestamp in Table of Contents links: integer seconds — e.g. `?t=154`
- Tables: use only when data has clear row/column structure. Never force prose into a table.
- Key Takeaways: 3-7 bullets, each starting with a noun or verb, no sub-bullets
- No emojis anywhere in the document
- Language: always match the transcript language
